# 📊 Portal System Sales - Inventory & Sales Management

## 🚀 Project Overview
Portal System Sales is a robust web-based solution designed for small to medium businesses to manage their daily operations. The system focuses on inventory control, sales tracking, and customer management, providing a clean interface for administrators to monitor business performance and generate transaction records efficiently.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Backend:** PHP
- **Database:** MySQL
- **Hosting/Server:** InfinityFree (Apache)
- **Architecture:** MVC (Model-View-Controller) pattern approach

## ⚙️ Features
- **Inventory Management:** Full CRUD for products, including stock alerts and category organization.
- **Sales Module:** Streamlined process for recording sales and generating transaction history.
- **User Authentication:** Secure login system for authorized personnel.
- **Client Database:** Manage customer information linked to sales records.
- **Reporting:** Quick view of total sales and stock levels.
